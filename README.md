Анализ данных для агентства недвижимости (Санкт-Петербург и ЛО) 

Проект посвящен анализу объявлений о продаже жилья для поиска наиболее перспективных сегментов рынка

Основные выводы: 1. Быстрее всего (в течение месяца) продаются однокомнатные квартиры во всех регионах.
2. Самое большое количество объявлений и высокая скорость снятия наблюдаются в Мурино, Кудрово и Шушарах.
3. Средняя стоимость метра в Санкт-Петербурге (108–130 тыс. руб.) значительно выше, чем в области (66–72 тыс. руб.).
4. Пик активности публикаций приходится на февраль и март.
